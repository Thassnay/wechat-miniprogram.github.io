# KFlex 

Flexbox 布局标签

## 使用指南

```js
import {KFlex, KFlexItem } from 'kbone-ui'
```

## 示例

```html
<KFlex>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
</KFlex>
```
