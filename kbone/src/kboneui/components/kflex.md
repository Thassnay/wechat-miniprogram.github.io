# KFlex 

Flexbox 布局标签

## 使用指南

```
import {KFlex, KFlexItem } from 'kbone-vue-weui'
```

## 示例

```
<KFlex>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
    <KFlexItem><KView class="placeholder">weui</KView></KFlexItem>
</KFlex>
```
